# pmi-2011-curriculum
https://kiri11.ru/curriculum-open-source/

Когда 8 апреля 2011 меня опять 4 человека спросили в аське, к какой нам паре, я решил написать предельно простую программу, которая отвечала бы на вопросы по расписанию (примерно в том же стиле, как я ответил бы в аську). Только для группы ПМИ-22.

Итак, качайте:

[Расписание 0.2, бинарник под винду (вирусов нет, отвечаю!)](build/pmi22.exe)
